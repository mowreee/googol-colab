## Getting Started

### Option 1: Using Jupyter Notebook Locally

To get started with this repository locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/mowreee/googol-colab.git
   ```
2. Navigate to the project directory:
   ```bash
   cd googol-colab
   ```
3. Open the Jupyter Notebook in your environment:
   ```bash
   jupyter notebook
   ```

### Option 2: Using Google Colab

If you'd like to use Google Colab:

1. Open [Google Colab](https://colab.research.google.com/).
2. Click on "File" > "Open Notebook" > "GitHub."
3. Enter the repository URL: `https://github.com/mowreee/googol-colab`
4. Select a notebook file to start working on it.
